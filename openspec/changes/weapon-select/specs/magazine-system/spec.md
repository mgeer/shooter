## ADDED Requirements

### Requirement: 弹夹容量限制
游戏中玩家 SHALL 持有有限弹夹，每次射击消耗 1 发（霰弹枪消耗 1 发但发射 5 颗子弹）。弹夹打空时无法继续射击，必须换弹后方可射击。弹药总量无限。

#### Scenario: 正常射击扣弹
- **WHEN** 玩家点击射击且当前弹数 > 0 且不在换弹中
- **THEN** 发射子弹，当前弹数减 1

#### Scenario: 弹夹为空无法射击
- **WHEN** 玩家点击射击且当前弹数 = 0
- **THEN** 不发射子弹，自动触发换弹

### Requirement: 换弹机制
系统 SHALL 支持自动换弹（弹夹打空时触发）和手动换弹（按 R 键）。换弹期间无法射击，换弹完成后弹夹恢复满弹。

#### Scenario: 弹夹打空自动换弹
- **WHEN** 玩家射击后当前弹数归零
- **THEN** 立即进入换弹状态，换弹时间由武器配置决定

#### Scenario: 按 R 手动换弹
- **WHEN** 玩家按下 R 键且不在换弹中且当前弹数 < 弹夹容量
- **THEN** 进入换弹状态

#### Scenario: 换弹中无法射击
- **WHEN** 玩家在换弹期间点击射击
- **THEN** 不发射子弹，换弹继续

#### Scenario: 换弹完成恢复满弹
- **WHEN** 换弹计时结束
- **THEN** 当前弹数恢复为弹夹容量，退出换弹状态

### Requirement: 弹药 HUD 显示
HUD SHALL 实时显示当前弹数与弹夹容量，换弹时显示换弹进度条。

#### Scenario: 正常显示弹药数
- **WHEN** 游戏进行中
- **THEN** HUD 显示"当前弹数 / 弹夹容量"（如 "6 / 12"）

#### Scenario: 换弹时显示进度条
- **WHEN** 换弹状态激活
- **THEN** HUD 显示换弹进度条，从空到满随时间增长，完成后隐藏进度条
